1. 錢包
   Crypto-Arsenal 的錢包目前僅用來支付訂閱費和分潤費。要投資的本金請留在交易所。這邊有轉幣教學，我們提供每一個用戶專屬的轉帳錢包（採用 CyBAVO 企業級多重加密錢包，安全無虞）。

2. KYC
   目前不需要 KYC，不影響平台使用。

3. 合約交易所
   合約交易支援 Binance、OKX 和 Bitget。

4. 創建 API Key 並複製、貼上到我們平台
   參考 Crypto-Arsenal 使用指南 API 或是教學影片
